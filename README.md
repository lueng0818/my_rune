# 🔮 北歐盧恩符文數位諮詢師 (Nordic Rune Digital Consultant)

> 連結古老智慧，透過數位儀式指引迷途。
> 基於《北歐盧恩符文證書課程》與《諮詢師整合班》講義開發。

## 📖 專案簡介

這是一個基於 **Streamlit** 開發的互動式網頁應用程式。本系統不僅僅是隨機抽牌程式，而是將盧恩符文（Runes）的**占卜儀式**、**奧丁禱詞**以及**專業牌陣解讀**數位化。

系統設計模擬了真實的諮詢流程，強調「靜心」與「神聖性」，並根據抽出的符文（包含正逆位）提供針對「事業」、「健康」與「情感」的詳細指引。

## ✨ 核心功能

* **🧘 靜心儀式 (Ritual)**：引導使用者進行身心放鬆，並誦讀奧丁禱詞，建立連結。
* **🃏 多種牌陣 (Spreads)**：
    * **單張指引**：適用於每日運勢或簡易問答。
    * [cite_start]**三張牌陣**：包含「時間流（過去/現在/未來）」與「行動指引（問題/作法/結果）」[cite: 566-607]。
    * [cite_start]**五張牌陣**：全方位解析，包含過去成因、未來發展、建議與挑戰 [cite: 608-635]。
* **🔄 自動逆位處理**：程式會隨機決定正逆位，並自動將符文圖片旋轉 180 度顯示。
* **📚 符文圖書館**：內建 25 個符文的詳細釋義（含事業、健康建議）。
