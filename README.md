# 🔮 北歐盧恩符文數位諮詢師 (Nordic Rune Digital Consultant)

> [cite_start]**「盧恩符文是活生生的無形老師，而非單純的算命工具。」** [cite: 21]
> [cite_start]本系統嚴格依據 **盧恩女神 Anne 老師** 的教學體系開發，旨在協助使用者連結古老智慧，在自己身上找到答案 [cite: 543-544]。

## 📖 專案簡介

[cite_start]這是一個基於 **Streamlit** 開發的互動式諮詢系統。不同於一般的隨機抽牌程式，本專案將盧恩符文（Runes）視為一套完整的生命指引系統，透過數位化的方式，重現神聖的諮詢體驗。系統設計的核心哲學在於：「神的旨意並非告知未來，而是從各種變數中指引你，讓你針對眼前問題，自己做最後的決定。」 [cite: 543-547]

## 🌌 系統架構與核心哲學

### 1. 起源與結構
* [cite_start]**系統起源**：源自西元前二世紀的古老智慧，由天地間大自然的元素與神族代表融合而成 [cite: 18]。
* [cite_start]**組成結構**：全套共 **25 張**，包含 24 個字母符文與 1 張空牌（宇宙牌 Wyrd） [cite: 18]。
* [cite_start]**三大群組**：依據時間表分為弗雷群組（生存）、海姆達爾群組（考驗）、提爾群組（精神） [cite: 31]。

### 2. 占卜儀式流程 (The Ritual)
本系統還原了講義中的標準儀式，確保諮詢的神聖性：
1.  [cite_start]**身心準備**：身體放鬆，心靈淨空（靜心） [cite: 550-551]。
2.  [cite_start]**祈禱詞**：呼喚全能且有智慧的奧丁神（Odin），請求指引手與意念以得到真理 [cite: 553-555]。

## 🗂️ 多維度符文資料庫 (The Database)

系統內建完整的符文釋義，每張牌均包含以下維度的解析：

* [cite_start]**日期對應**：每張牌對應特定的半個月日期區間（例如：紫杉 Eihwaz 對應 12/28-1/12） [cite: 36]。
* **身心靈狀態**：
    * [cite_start]**正位**：如堅持不懈、正向改變 [cite: 39, 305]。
    * [cite_start]**逆位**：如瓶頸、停滯、需內部轉化 [cite: 138, 462]。
* [cite_start]**身體部位**：用於健康占卜（例如：紫杉對應脊椎 [cite: 41][cite_start]、聖杯對應婦科/子宮 [cite: 69][cite_start]、雷神對應肝臟 [cite: 327]）。
* [cite_start]**產業對應**：指出適合的職業（例如：聖杯對應女性/博弈產業 [cite: 71-73][cite_start]、紫杉對應公務員 [cite: 43]）。
* [cite_start]**愛情特質**：描述伴侶性格（例如：戰神牌代表大男人、霸道 [cite: 134]）。

## 🃏 專業牌陣系統 (Spreads)

依據問題深度與類型，提供不同等級的牌陣選擇：

### 1. 單張牌 (Daily/Instant)
* [cite_start]**用途**：每日運勢、未知狀況指引或詢問失聯人事物 [cite: 556-565]。

### 2. 三張牌 (Process/Choice)
* [cite_start]**是非題**：判斷「是否應該...」（如辭職、結婚） [cite: 574-575]。
* [cite_start]**適配度**：判斷物件（如水晶）是否適合 [cite: 579-580]。
* **線性發展**：1. [cite_start]問題出現（核心）→ 2. 採取作法 → 3. 結果 [cite: 596-607]。

### 3. 五張牌 (Holistic)
* **結構**：1. [cite_start]過去 → 2. 現在 → 3. 未來 → 4. 幫助/建議 → 5. 問題/挑戰 [cite: 609-628]。
* [cite_start]**邏輯**：透過第 4 張的「建議」來解決第 5 張的「挑戰」，進而影響未來 [cite: 629-634]。

### 4. 七張牌 (Detailed Analysis)
* [cite_start]**用途**：深入分析困境成因，時間涵蓋過去與未來各三個月 [cite: 636-639]。
* [cite_start]**結構**：包含問題現狀、過去因素（2張）、忠告（2張）、結局 [cite: 643-674]。

### 5. 時間流牌陣 (Yearly)
* [cite_start]**用途**：預測未來幾年的特定運勢（如離職、創業），每年抽一張牌代表 [cite: 1037-1052]。

## 🧠 解讀邏輯與原則

### 空牌 (Wyrd/宇宙牌)
* [cite_start]代表因果業力、命運安排 [cite: 484, 692]。
* [cite_start]若出現在「結果」位置，表示問題此刻無定論，需聽從命運指引，通常需補抽一張配牌 [cite: 485, 698]。

### 組合解讀 (配牌效應)
* [cite_start]**緩和**：單張牌義會因旁邊的牌而改變（例如：豐收 Jera + 負面 Rune = 過去挫敗因勤奮得到緩和） [cite: 471]。
* [cite_start]**代價**：颶風 (Hagalaz) + 正面 Rune = 冒險會成功但需艱苦奮鬥 [cite: 803]。

## 💡 主題式應用詳解

### 🏥 健康占卜
* [cite_start]**邏輯**：找出對應的身體部位與問題根源（如：抽到雷神代表肝火/過勞） [cite: 327, 929]。
* [cite_start]**建議**：透過符文給予作法（如：抽到水牌 Laguz 建議多做水療法、清理淨化） [cite: 930]。

### ❤️ 愛情占卜
* [cite_start]**二擇一牌陣**：為對象 A 與 B 各抽一張牌看適合度，最後抽一張建議牌 [cite: 939-940]。
* [cite_start]**特質判斷**：例如戰神 (Tiwaz) 代表意圖純正但也可能霸道 [cite: 134][cite_start]；保護 (Algiz) 代表忠誠與母性之愛 [cite: 102-103]。

### 💼 事業占卜
* [cite_start]**決策牌陣**：用於創業或合夥的二擇一選擇 [cite: 1054]。
* [cite_start]**產業對應**：例如財富 (Fehu) 對應金融精品 [cite: 288][cite_start]、神駒 (Ehwaz) 對應貿易運輸 [cite: 170]。

## 🦋 盧恩與女神牌卡整合應用

根據講義的進階應用，建議使用者在使用本系統後，可結合 **女神牌卡 (Goddess Oracle Cards)** 進行互補：

1.  **互補功能**：
    * **盧恩符文（骨架）**：負責揭示現實層面的狀況、吉凶、時間點、身體部位或外部環境結構。
    * [cite_start]**女神牌卡（血肉）**：負責揭示內在心靈層面、潛意識恐懼、陰性力量的運用以及靈性建議 [cite: 937, 964]。
2.  **實務操作**：
    * 先使用盧恩符文分析客觀局勢（過去/現在/未來）。
    * [cite_start]再搭配 1 到 3 張女神牌，給予案主身心平衡與內在轉化的建議 [cite: 953, 1010]。
3.  [cite_start]**諮詢原則**：諮詢師應保持中立，將決定權交還給個案，並引導個案覺察身心狀況 [cite: 1008-1009]。
